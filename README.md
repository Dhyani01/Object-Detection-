# Object-Detection-
Yolov3 based Algorithm that uses deep convolutional neural networks to perform object detection.
 
 
 
 
Dependencies:
    - python=3.7
    - pip>=19.0
    - jupyter
    - tensorflow-gpu=2.1.0
    - scikit-learn
    - scipy
    - pandas
    - pandas-datareader
    - matplotlib
    - pillow
    - tqdm
    - requests
    - h5py
    - pyyaml
    - flask
    - opencv

 Will add another route for labeling images for your custom dataset in future.
